# 更新日志

本项目采用语义化版本号：`主版本.次版本.修订版本`。

## 0.1.0 - 2026-07-22

第一个公开预览版本。

- 支持 macOS 原生应用构建。
- 支持荣耀 / 华为实况 JPG 检测与播放。
- 支持苹果 Live Photo 常见的 `HEIC + MOV` 同名配对识别。
- 支持扫描 HEIC/HEIF 图片。
- 支持照片精选、标签、搜索、历史目录和筛选。
- 支持展示 EXIF 元信息：设备型号、拍摄时间、像素尺寸；GPS 会尽量反解析成地点名称，不直接展示裸坐标。
- 支持在主窗口内查看照片，不再使用照片弹窗。
- 支持通过胶片缩略条选择实况照片的封面 / 停留帧。
- 实况播放结束后默认回到封面图，不再停留在视频最后一帧。
- 支持导出原始照片文件。
- 导出和安卓同步会一并带上照片的同名 MOV/MP4 sidecar 文件。
- 支持通过 adb 将精选照片同步到手机 `DCIM/MotionAlbum`。
- 提供 macOS Release 打包脚本，可生成 `.zip`、SHA256 校验文件；环境支持 `hdiutil` 时也会生成 `.dmg`。
- 保留 Windows WPF 预览版源码，暂不作为正式版本发布。
