**开发者**: 神马都会亿点点的毛毛张  
**版本**: 1.0.0

## 功能介绍

专为荣耀/华为手机用户打造的**实况照片查看器**，解决 Windows 通过 MTP 连接手机时只能看到静态 JPG、无法读取动态视频的问题。

### 核心功能

- 🖼️ **缩略图浏览** - 选择文件夹后，以卡片网格形式浏览所有照片
- 🎬 **实况视频播放** - 双击照片即可查看大图，点击"播放实况"按钮播放嵌入的动态视频
- ⭐ **收藏标注** - 对喜欢的照片点击星标收藏，支持"只显示收藏"筛选模式
- 🔄 **便捷切换** - 大图模式下支持键盘左右方向键、上一张/下一张按钮快速切换
- 💻 **内存友好** - 播放时采用逐帧读取技术，不预加载整个视频到内存

## 环境配置

本项目使用 Anaconda 虚拟环境 `livephoto_viewer`。

### 已安装的依赖

```bash
conda activate livephoto_viewer
pip list
```

主要依赖：
- Python 3.11
- PyQt5
- Pillow
- imageio + imageio-ffmpeg
- numpy

### 一键启动

双击项目根目录下的 `run.bat`：

```
run.bat
```

或手动在命令行中运行：

```bash
conda activate livephoto_viewer
python -m livephoto_viewer.main
```

## 使用说明

### 1. 浏览照片

启动程序后，点击顶部 **"📂 选择文件夹"** 按钮，选择存放手机照片的目录（如从手机复制到电脑的 `DCIM/Camera` 文件夹）。

程序会自动：
- 扫描所有 `.jpg` 文件
- 检测哪些是包含实况视频的动态照片（右上角显示 `LIVE` 红色角标）
- 生成缩略图预览

### 2. 播放实况视频

双击任意照片进入大图查看模式：

- 如果是**实况照片**，中央会显示 **"▶ 播放实况"** 按钮，点击即可播放嵌入的动态视频
- 如果是**普通照片**，直接显示大图，无播放按钮
- 播放完一次后自动停止，可点击 **"↻ 重播"** 再次播放

> **快捷键**: 空格键播放 / 左右方向键切换照片 / ESC 关闭查看器

### 3. 收藏标注

- 在缩略图网格中，点击照片右上角的 **☆** 星标按钮即可收藏，再次点击取消
- 点击顶部工具栏的 **"☆ 只显示收藏"** 按钮，可快速筛选出已收藏的照片
- 收藏状态会自动保存，下次打开同一文件夹时仍然有效

### 4. 批量提取视频（命令行工具）

如果只想把实况视频批量提取为独立的 `.mp4` 文件，可使用附带的提取工具：

```bash
conda activate livephoto_viewer
python honor_livephoto_extractor.py -b "你的照片文件夹路径"
```

## 技术原理

### 为什么 Windows 看不到实况视频？

荣耀/华为手机的动态照片采用了一种**隐藏式嵌入**存储方案：

1. 拍摄时同时保存：静态封面图（标准 JPEG）+ 动态视频（完整 MP4，H.265 + AAC）
2. MP4 的 `ftyp` 标记位于 JPG 文件数据的**中间位置**（约第 4~6MB 处）
3. Windows / MTP 协议只读取到 JPEG 的结束标记 `FF D9`，根本不会扫描文件内部的 `ftyp`

### 本工具如何提取？

1. 在 JPG 文件中全局搜索 `ftyp` 字符串
2. 向前回退 4 字节（MP4 box size 字段）
3. 从该位置截取到文件末尾，即为完整的 MP4
4. 播放时采用 `imageio` 库逐帧读取，内存占用极小

## 常见问题

**Q: 播放实况视频时卡顿？**
A: 实况视频是 H.265 (HEVC) 编码，首次播放时需要从 JPG 中临时提取到磁盘，这个过程大约几百毫秒。如果持续卡顿，请确保电脑有足够的磁盘空间用于临时文件。

**Q: 收藏的数据保存在哪里？**
A: 收藏状态保存在 `%APPDATA%/LivePhotoViewer/favorites.json` 中，按文件夹路径分类存储，即使移动照片文件夹位置也不会丢失。

**Q: 能否把实况照片发给 iPhone 用户并保留动态效果？**
A: iPhone 的 Live Photo 格式与安卓 Motion Photo 不同。如果要发给 iPhone 用户，建议先提取为 MP4 发送，或转换为 MOV 格式。后续版本可能会增加导出为 iPhone 兼容格式的功能。
